Topic: Seasonal Demand Forecasting and Personalized Product Recommendation for Inventory Optimization in Retail Business
Project Description: This project combines Seasonal Demand Forecasting and Personalized Product Recommendation to optimize inventory management in the retail business domain.
It is designed to run directly on Google Colab, enabling easy setup without local environment configuration.
Objectives:
Forecast product demand based on seasonal patterns
Provide personalized product recommendations for customers
Optimize inventory levels to reduce overstock and stockout risks

How to Run on Google Colab
1. Open Google Colab [https://colab.research.google.com/](url)
2. Upload Notebook STAT3013.Q21.Group03_DemandForecast_SourceCode_20260429.ipynb
3. Install Dependencies !pip install numpy pandas scikit-learn matplotlib seaborn statsmodels lightgbm xgboost
4. Upload dataset STAT3013.Q21.Group03_DemandForecast_Dataset_20260429.csv

Dataset link: [https://www.kaggle.com/datasets/utkalk/large-retail-data-set-for-eda](url)
